VS Code 拡張機能

コードを、段階的に
読み解く。

Unfoldは、AIがコードを論理ブロックごとに区切り、順番に解説するVS Code拡張機能です。 コードを書くのではなく、読んで理解する過程を支援します。

Features

主な特徴

Unfoldは「コードを書くAI」ではなく、コードの読み方・理解の仕方を支援するツールです。

ブロック単位で順に読める

コードを意味のあるまとまりに分割し、ひとつずつ順番に解説します。

現在地とコードが対応する

今読んでいる解説がコード上のどこに対応しているか、常にわかります。

必要な場所だけ深掘りできる

わかっている部分はスキップし、気になる箇所だけドリルダウンで詳しく確認できます。

初学者でも追いやすい

専門用語を前提にしない段階的な構成で、学び始めた方にも使いやすい設計です。

Who is it for

こんな人に向いています

プログラミング初学者

コードの読み方にまだ慣れていなくても、ブロックごとに順を追って理解を進められます。

AIが書いたコードを
理解したい人

新しいプロジェクトに
入った人

How to use

使い方

3つのステップで、すぐに使い始められます。

1

拡張機能をインストール

VS Code Marketplace で「Unfold」を検索し、インストールします。

2

APIキーを設定

設定画面からご自身のAPIキーを入力します。BYOK方式です。

3

ファイルを開いて解説開始

コマンドを実行すると、ブロック単位の解説が始まります。

Notice

ご利用にあたって

現時点での注意事項

  • BYOK（Bring Your Own Key）方式のため、ご自身のAPIキーが必要です。
  • 現在MVP版です。対応する言語や機能に制限があり、すべてのケースで完璧に動作するとは限りません。
  • 今後のアップデートで機能や仕様が変更される可能性があります。フィードバックを歓迎しています。
FAQ

よくある質問

Unfoldは何をするツールですか？

コードを論理ブロックに分割し、順番に解説するVS Code拡張機能です。コードを「読んで理解する」ことを支援します。

APIキーは必要ですか？

はい。BYOK方式のため、ご自身のAPIキーをご用意ください。詳細はMarketplaceのページをご参照ください。

どんな人に向いていますか？

初学者、AIが生成したコードを理解したい方、新しいプロジェクトで既存コードを読む必要がある方に向いています。

どのプログラミング言語に対応していますか？

MVP版のため対応言語に制限があります。詳細はMarketplaceのページをご確認ください。

利用料金はかかりますか？

Unfold自体は無料です。APIの利用料金はお使いのプロバイダーの料金体系に準じます。
