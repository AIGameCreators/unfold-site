Unfold

AIがコードを論理ブロックごとに分けて、段階的に読み解きを支援する VS Code 拡張です。

VS Code Marketplaceで見る

※ リンク先は公開後に差し替えてください

できること

  • コードをブロック単位で順に読む
  • 現在読んでいる説明とコード位置を対応表示する
  • 必要な箇所だけドリルダウンして深掘りする

対象ユーザー

プログラミング初学者、新しいコードベースに入った開発者、AIが書いたコードを後から理解したい人向けです。

使い方

  1. VS Code に拡張をインストール
  2. APIキーを設定
  3. コードファイルを開いて解説を開始

注意点

  • BYOK（自分のAPIキーを使う）方式です
  • MVP版のため、対応言語や機能には制限があります
